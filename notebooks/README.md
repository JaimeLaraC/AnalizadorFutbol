# Notebooks - AnalizadorFutbol

Jupyter notebooks para exploración de datos y entrenamiento de modelos.

## Contenido

1. `01_api_exploration.ipynb` - Exploración de API-Football
2. `02_data_analysis.ipynb` - Análisis exploratorio de datos
3. `03_feature_engineering.ipynb` - Creación de features
4. `04_model_training.ipynb` - Entrenamiento de modelos
5. `05_predictions.ipynb` - Generación de predicciones

## Ejecución

```bash
cd backend
source venv/bin/activate
jupyter notebook
```
